Vie et Carrière 3D

Un jeu 3D sur le web inspiré de l'univers de François Pérusse. Vous incarnez un personnage traversant différentes étapes de la vie (école primaire, école secondaire, cégep/université, marché du travail) en évitant des obstacles et en ramassant des objets (bulletins, bourses, diplômes) pour accumuler des points et vous rendre jusqu'à la radio CDKC !

Fonctionnalités

Jeu de course 3D : Esquivez les obstacles et sautez par-dessus les dangers de la vie.

Étapes de la vie : Traversez différentes phases avec leurs propres défis (école primaire, secondaire, université, etc.).

Dialogues et Humour : Citations et clins d'œil à l'œuvre de François Pérusse.

Contrôles adaptatifs : Jouable au clavier sur ordinateur ou via des contrôles tactiles sur mobile.

Technologies utilisées

React : Interface utilisateur (HUD, menus, dialogues).

Three.js : Rendu 3D du jeu, gestion de la caméra, des modèles et des collisions.

Tailwind CSS : Styles de l'interface utilisateur.

Vite : Outil de build et serveur de développement.

Comment jouer

Sur ordinateur (Clavier)

Flèche Gauche / A : Se déplacer vers la gauche

Flèche Droite / D : Se déplacer vers la droite

Espace / Flèche Haut / W : Sauter

X / Entrée : Lancer un objet (tirer)

Sur mobile (Tactile)

Des boutons virtuels s'affichent automatiquement au bas de l'écran (Gauche, Droite, Saut, Tir).

Déploiement

Ce projet est prêt à être déployé sur n'importe quel hébergement statique (Vercel, Netlify, GitHub Pages, Firebase Hosting, etc.).

Clonez le dépôt

Installez les dépendances : npm install

Compilez le projet : npm run build

Le dossier dist/ contiendra les fichiers prêts pour la production.

Contribuer

N'hésitez pas à ouvrir des issues ou soumettre des pull requests pour ajouter de nouvelles étapes de vie, des répliques, ou pour améliorer les graphismes et la jouabilité !
